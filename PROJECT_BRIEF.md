# Aave Risk Signals

Subtitle: Forward yield and reserve stress signals for Aave.

## Goal
Build a small demo app for the functionSPACE competition.
The app should look like a prediction-market-style terminal for forward-looking Aave signals.

## Scope for MVP
- Frontend only
- Mock data only
- No wallet
- No backend
- No real blockchain trades yet
- Clean architecture for later functionSPACE integration

## Core market types
1. Forward yield signal markets
2. Reserve stress signal markets

## Product principles
- Clean and credible
- Not overbuilt
- Easy to demo
- Market definitions configurable from a single file
- Adapter layer for future Aave data + functionSPACE SDK integration

## Important
This repo should be optimized for speed of iteration, not production readiness.
